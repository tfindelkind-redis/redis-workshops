# 2-Day Deep Dive Workshop: Azure Managed Redis - Complete Plan

**Workshop Type:** Configuration 4 - 2-Day Deep Dive  
**Duration:** 12 hours content + 3 hours breaks = 15 hours total  
**Target Audience:** Architects, Senior Developers, Platform Engineers  
**Prerequisites:** Azure experience, Redis fundamentals, programming (Python/C#/.NET)

---

## Workshop Overview

This comprehensive 2-day workshop provides deep technical coverage of Azure Managed Redis, with extensive focus on the Well-Architected Framework, hands-on labs, and production-ready implementation patterns.

### Learning Objectives

By the end of this workshop, participants will be able to:

1. **Design** production-ready Redis architectures using Azure Managed Redis
2. **Apply** Well-Architected Framework principles across all 5 pillars
3. **Implement** caching patterns and data modeling best practices
4. **Deploy** Redis infrastructure using IaC (Bicep/Terraform)
5. **Configure** monitoring, alerting, and security controls
6. **Troubleshoot** common issues and perform migrations
7. **Leverage** advanced Redis features (modules, vector search, streams)

### Workshop Structure

**Day 1 (6.5 hours):** Foundation, Architecture & Well-Architected Framework  
**Day 2 (5.25 hours):** Hands-On Labs & Advanced Topics
