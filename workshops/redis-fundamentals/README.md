# Redis Fundamentals

Welcome to the Redis Fundamentals workshop! This workshop will guide you through the core concepts of Redis, from basic data structures to practical use cases.

## Workshop Overview

Redis (REmote DIctionary Server) is an open-source, in-memory data structure store used as a database, cache, message broker, and streaming engine. In this workshop, you'll learn:

- What Redis is and why it's used
- Core Redis data structures
- Essential Redis commands
- Real-world Redis patterns and use cases

## Modules

### Module 1: Introduction to Redis (45 minutes)
Learn what Redis is, its architecture, and when to use it in your applications.

**Topics Covered:**
- What is Redis?
- Redis vs other databases
- Key Redis features
- Common use cases

### Module 2: Redis Data Structures (75 minutes)
Deep dive into Redis data structures with hands-on exercises.

**Topics Covered:**
- Strings: Basic key-value operations
- Lists: Queues and stacks
- Sets: Unique collections
- Hashes: Structured objects
- Sorted Sets: Ordered collections with scores

### Module 3: Redis Use Cases (60 minutes)
Apply your Redis knowledge to solve real-world problems.

**Topics Covered:**
- Caching strategies
- Session management
- Pub/Sub messaging
- Leaderboards and counters
- Rate limiting

## Prerequisites

- Basic understanding of databases
- Familiarity with command line interfaces
- No prior Redis experience required

## Setup

Ensure you have:
- Redis installed (version 6.0 or higher)
- Redis CLI access
- Optional: RedisInsight for visualization
